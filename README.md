# Project Jammming

## Goal of the project:

The goal of the project is to allow users to search the spotify library, create a custom playlist and then save it to their spotify-account.

## Technology used:

- HTML
- CSS
- Javascript
- REACT framework
- HTTP Requests and Responses
- Authentication

## Features:

- Users can search for songs by song title
- Users can see information about each song (like artist,genre,name etc.)
- Users can export their custom playlist to their personal Spotify account
